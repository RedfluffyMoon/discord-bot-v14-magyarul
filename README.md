## 🚀 Telepítés

### 📋 Követelmények

- [Node.js](https://nodejs.org/) v18+
- [npm](https://www.npmjs.com/)
- Discord Bot Token ([Discord Developer Portal](https://discord.com/developers/applications))
- OpenWeatherMap API kulcs (csak `/weather`-hez)

### 1️⃣ Repository klónozása

```bash
git clone https://github.com/RedfluffyMoonSzia/discord-bot-magyarul.git
cd discord-bot-magyar
```

### 2️⃣ Csomagok telepítése

```bash
npm install
```

### 3️⃣ .env fájl létrehozása

```env
TOKEN=a_te_bot_tokenedXXXXXXX
CLIENT_ID=a_te_bot_idXXXXXXX
GUILD_ID=a_te_szerver_idXXXXXXX
WEATHER_API_KEY=openweathermap_api_kulcsXXXXXXX
```

### 4️⃣ Parancsok regisztrálása

```bash
node deploy-commands.js
```

### 5️⃣ Bot indítása

```bash
node index.js
```

## 🌐 Integrációk

### 🟣 Replit
<a href="https://replit.com/github/RedfluffyMoonSzia/discord-bot-magyarul">
  <img src="https://replit.com/badge/github/RedfluffyMoonSzia/discord-bot-magyarul" alt="Run on Replit" height="40"/>
</a>

> 1. Kattints a gombra
> 2. Replit-en nyisd meg a **Secrets** menüt (🔒 ikon)
> 3. Add hozzá a `.env` változókat:
>    - `TOKEN`
>    - `CLIENT_ID`
>    - `GUILD_ID`
>    - `WEATHER_API_KEY`
> 4. Kattints a **Run** gombra ▶️

### 🚂 Railway
[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)

> 1. Kattints a gombra
> 2. Kösd össze a GitHub fiókodat
> 3. Add meg a környezeti változókat
> 4. Deploy! 🚀

### 🐳 Docker

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
CMD ["node", "index.js"]
```

```bash
docker build -t discord-bot-magyar .
docker run -d --env-file .env discord-bot-magyar
```
